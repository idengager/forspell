# Forspell
Forspell is a spelling checker for your code documentation.

Supports C, C++, Ruby and Markdown files.

## Requirements
* [libhunspell] >= 1.2.0
* [ffi] ~> 1.0

# Installation

`gem install forspell`

# Usage

`forspell lib *.md`

[libhunspell]: http://hunspell.sourceforge.net/
[ffi]: https://github.com/ffi/ffi
